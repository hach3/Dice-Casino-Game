# Dice Casino Game

## About / Synopsis

Dice Casino Game is a project made by Nebuchadnezzar.dev. It uses its API to provide a casino game with a custom RTP

## Installation

Sample:

* git clone https://github.com/hach3/Dice-Casino-Game.git
* cd Dice-Casino-Game
* npm i
* node index
* Visit http://localhost:8080

### Requirements

 * NodeJS
 * NPM
